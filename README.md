# 통합재난경보발령시스템 #

우리의 프로젝트에서는 대한민국 통합재난경보발령시스템(Korea Integrated Emergency Alert System)의 발전을 위한 참조모델의 연구개발이 진행중에 있다. 당신은 이 페이지에서 우리의 프로젝트를 이해하기 위해 필요한 정보들을 얻을 수 있다.
